Добавьте в систему обработки транзакций из предыдущего задания
пятое правило для автоматического обнаружения мошенничества.

Пятое правило для автоматического обнаружения мошенничества:
- Rule 5: все транзакции трейдеров из Германии на сумму
  больше 1000 должны быть отклонены.

Для реализации этого правила внесите в код класса FraudDetector
любые изменения, которые вы считаете нужными.

Добавьте в класс FraudDetectorTest тесты,
которые доказывают, что ваша реализация пятого правила
по определению мошеннических транзакций в классе
FraudDetector работает правильно!

При написании тестов не забудьте протестировать
позитивные и негативные сценарии.
