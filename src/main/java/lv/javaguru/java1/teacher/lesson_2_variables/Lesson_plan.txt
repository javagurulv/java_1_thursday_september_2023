Введение в понятие данных и алгоритмов.

Познакомить слушателя с простыми (примитивными)
типами данных (int, long, float, double, boolean, char).

Объяснить понятие переменной (временное хранение промежуточных
результатов вычисления).

Объявление переменной, инициализация,
присваивание значения, использование значения.

Научиться быстро создавать новые классы для написания маленьких программ.

Научиться выполнять основные операции с Git: add, commit, push, pull.
