Добавить в Git созданные ранее классы и
познакомиться с командами Git->commit, Git->push.

Цели:
- научиться добавлять файлы в Git (зелёные, команда Git->Add);

- научиться комитить изменения в локальный репозиторий
  Git (команда Git->Commit);

- научиться пушить изменения в глобальный репозиторий
  (GitHub) (Git->Push).
