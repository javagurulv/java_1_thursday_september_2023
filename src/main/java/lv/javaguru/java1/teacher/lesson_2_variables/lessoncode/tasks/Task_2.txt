Написать программу, которая печатает
ваше имя в одной строке и фамилию в другой строке на консоль.

Цели:
- научиться копировать классы без использования
  мыши (Ctrl + C -> Ctrl + V);

- научиться давать классам новые названия
  (CamelCase нотация https://ru.wikipedia.org/wiki/CamelCase);

- познакомиться с последовательностью команд,
  каждая команда в новой строке.
