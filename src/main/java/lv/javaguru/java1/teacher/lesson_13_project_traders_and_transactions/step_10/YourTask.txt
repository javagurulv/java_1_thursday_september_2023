Реализация метода для нахождения уникальных имён трейдеров
находится тут:
TransactionAnalysisService.findUniqueTraderNames().


Ваша задача: напишите метод, который принимает на вход список транзакций
и находит все уникальные города в которых работают трейдеры.

Реализацию этого метода напишите с использованием Stream API.

В классе TransactionAnalysisServiceTest напишите тест для созданного вами метода.

Для тестирования используйте тестовые данные из класса TransactionTestData.
