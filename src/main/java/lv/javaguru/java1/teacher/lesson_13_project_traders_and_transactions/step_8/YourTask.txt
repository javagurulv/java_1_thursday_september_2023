Реализация метода для нахождения всех годов в которых были
совершены транзакции находится тут:
TransactionAnalysisService.findTransactionYears().


Ваша задача: найти из всех транзакций уникальные года в которые
они были совершены.

Реализацию этого метода напишите с использованием Stream API.

В классе TransactionAnalysisServiceTest напишите тест для созданного вами метода.

Для тестирования используйте тестовые данные из класса TransactionTestData.
