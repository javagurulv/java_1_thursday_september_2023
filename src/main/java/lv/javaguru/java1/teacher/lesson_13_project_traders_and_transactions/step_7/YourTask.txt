Метод для нахождения всех транзакций совершённых в 2011 году
+ сортировка найденных транзакций по параметру value от меньшего к большему:

TransactionAnalysisService.findTransactionsByYearAndSortByValueASC().

Ваша задача: в классе TransactionAnalysisService
напишите метод, который принимает на вход список транзакций
и находит все года, в которых были совершены транзакции (года могут повторяться).

Реализацию этого метода напишите с использованием Stream API.

В классе TransactionAnalysisServiceTest напишите тест для созданного вами метода.

Для тестирования используйте тестовые данные из класса TransactionTestData.
