Создать маленькое приложение для получения информации по клиентам.

Представьте вы работаете в банке. Банк хочет дать возможность
внешним приложениям запрашивать данные о клиентах банка.
Вам поручили разработать прототип этого решения.

Для обеспечения безопасности клиенты должны в запросе предоставлять
специальный ключ, который им выдаёт банк при заключении договора
о пользовании этим API.

В запросе так же должен присутствовать id клиента данные которого
приложение запрашивает.

Вам надо смоделировать интерфейс этого решения, и реализовать его
прототип используя фейковые клиентские данные хранящиеся в памяти
или в файле.
