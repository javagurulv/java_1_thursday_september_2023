Для управления собственными файлами создайте свой пакет
в папке src/main/java/lv/javaguru/java1.

Для этого нужно выбрать src/main/java/lv/javaguru/java1
в окне Project (Alt + 1), вызвать всплывающее меню
и выбрать команду: New -> Package и в появившемся
окне ввести имя пакета воспользовавшись шаблоном:
student_yourname_yoursurname.

Например: student_vasilij_ivanov.

У каждого слушателя будет своя уникальная папка
внутри нашего проекта, таким образом вы не будете мешать друг другу.

Внутри созданной вами папки вы сможете создавать и управлять
файлами ваших проектов.
