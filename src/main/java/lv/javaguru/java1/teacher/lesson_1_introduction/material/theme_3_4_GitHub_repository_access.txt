Для получения доступа к репозиторию на GitHub
вам необходимо сообщить ментору имя вашего пользователя на
GitHub. Ментор добавит вашего пользователя в коллабораторы
проекта. После добавления вашего пользователя в коллабораторы
проекта вам на емейл (с которым вы регистрировались на GitHub)
придёт от емейл от GitHub. В этом емейле вас попросят подтвердить
доступ к репозиторию. Подтвердите доступ! После этого у вашего
GitHub пользователя появятся права на просмотр и на запись
в репозиторий, который вы будете использовать во время занятий.
