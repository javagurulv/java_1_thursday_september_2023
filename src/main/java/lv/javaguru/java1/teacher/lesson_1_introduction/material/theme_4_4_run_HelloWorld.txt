Если вы все сделали правильно, то в около строк

public class HelloWorld {

и

public static void main(String[] args) {

появятся слева зелёные треугольники.

Нажмите на любой из них и выберите Run 'HelloWorld.main()'.
Это запустит программу, откроется окно консоли внизу,
на которой будет напечатано:

> Task :HelloWorld.main()
Hello from Java!


PS: Если вы не можете быстро найти в консоли Hello from Java!
воспользуйтесь функцией поиска текста в окне консоли. Для этого
поставьте курсов в окно консоли и нажмите Ctrl + F, в поле поиска
введите текст, который хотите найти.
