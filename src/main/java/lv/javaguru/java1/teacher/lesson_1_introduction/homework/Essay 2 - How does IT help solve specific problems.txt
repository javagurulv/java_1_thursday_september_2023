Возьмите пару примеров из вашего эссе
"Где в современном мире используется ИТ?"
и для каждого из них попробуйте:

- описать конкретный пример задачи которую помогает решать ИТ;
- ответить на вопрос: как ИТ решает эту задачу? (не знаете
  ответа на этот вопрос пофантазируйте);
- можно ли решить эту задачу в ручную (без ИТ программы)?;
- опишите преимущества и недостатки ИТ решения задачи и
  решения задачи в ручную.

Напишите короткое эссе в свободной форме на тему:

"Как ИТ помогает решать конкретные задачи?"

(1 - 2 страницы не более, сохраните его как файл с расширением txt)

Целью данного эссе является определение конкретных задач,
которые решает ИТ, изучение способов решения
этих задач при помощи ИТ, сопоставление ИТ решения и
решений задач в ручную.

Данная работа позволит вам углубиться в задачи, которые решает ИТ
и увидеть (понять/осознать/осмыслить) смысл использования ИТ решений
в сравнении их с решениями задач в ручную.
