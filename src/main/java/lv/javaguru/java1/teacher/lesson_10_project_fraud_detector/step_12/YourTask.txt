От нас требуют, чтобы наш код возвращал два результата:
- является транзакция мошенничеством или нет (boolean)
- название правила согласно которому транзакция признана
  мошенничеством (String), это поле заполняется только в том
  случае если транзакция является мошенничеством, иначе null.

Вернуть два разных по типу результата из метода
FraudDetector.isFraud(Transaction t) мы не можем
(синтаксис языка Java не позволяет этого сделать).

Решение может выглядеть следующим образом:
- создаём отдельный класс FraudDetectionResult
  для объединения двух возвращаемых значений;
- меняем класс FraudDetector так, чтобы из
  метода isFraud() возвращался FraudDetectionResult;
- меняем логику работы класса FraudDetector для
  возвращения правильного типа результата.

Набросок кода для класса FraudDetectionResult:

class FraudDetectionResult {

    private boolean fraud;
    private String ruleName;

}

предложенное и описанное выше решение!
