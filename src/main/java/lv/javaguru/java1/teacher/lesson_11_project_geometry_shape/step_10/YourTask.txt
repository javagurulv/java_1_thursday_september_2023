В классе ShapeUtil реализуйте метод:

    Shape createRandomShape()

Суть этого метода проста, этот метод случайным образом
должен создавать одну из геометрических фигур. Для
определения какую фигуру создать, воспользуйтесь
генератором случайных чисел (сгенерируйте случайное
целое число: 0 - круг, 1 - квадрат, 2 - прямоугольник).
