Легенда проекта.

Вы работаете разработчиком в компании
занимающейся проектированием.

В компании каждый день архитекторы занимаются проектированием
новых объектов (домов, квартир, офисных помещений, и т.д.).

В процессе проектирования возникают рутинные задачи,
которые необходимо автоматизировать.

Вам поручено написать программное обеспечение для
автоматизации рутинных задач.

Сеньор разработчик уже создал для вас первый
класс в коде. Пожалуйста, ознакомьтесь с ним.

Класс Shape будет представлять главный класс
иерархии геометрических фигур. Геометрические
фигуры будут представлять собой помещения
(например квадрат, прямоугольник, круг, треугольник, и т.д.).

Две рутинные задачи для автоматизации:
- расчёт площади помещения;
- расчёт периметра помещения.

Ваша первая задача: добавить в класс Shape
абстрактный метод для подсчёта периметра.
