Написать автоматические тесты на класс CompoundInterestCalculator
в котором реализован метод расчета формулы сложно процента.

Класс CompoundInterestCalculator лежит рядом с этим файлом задания.

Для выполнения задания скопируйте класс CompoundInterestCalculator
к себе в пакет.

Создайте класс CompoundInterestCalculatorTest и добавьте в него
необходимые тестовые методы.

Перед написанием тестов продумайте бизнес сценарии:
- что будет если sum = 0;
- что будет если interest = 0;
- что будет если years = 0;
Включите в свой набор автоматических тестов указанные выше сценарии.

Надёжный программный код должен работать правильно во всех случаях!
Залог этого - хорошо продуманные тестовые сценарии, в которых
протестирован не только happy path, но и граничные значения
(экстремальные случаи).

Цели:
- закрепить навык написания автоматических тестов;
- тренировка в генерации тестовых вариантов;
- повторение трёх шагов в написании автоматических тестов:
    подготовка тестовых данных,
    вызов продакшн кода,
    проверка результата.
