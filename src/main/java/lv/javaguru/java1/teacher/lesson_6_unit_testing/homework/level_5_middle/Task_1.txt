Написать автоматические тесты на класс ColourWaveDetector
в котором реализован метод преобразования длинны волны в название цвета.

Класс ColourWaveDetector лежит рядом с этим файлом задания.

Для выполнения задания скопируйте класс ColourWaveDetector к себе в пакет.

Создайте класс ColourWaveDetectorTest и добавьте в него
необходимые тестовые методы.

Должны быть протестированы все возможные варианты!
Каждый вариант должен быть протестирован отдельно друг от друга.

Цели:
- закрепить навык написания автоматических тестов;
- тренировка в генерации тестовых вариантов;
- повторение трёх шагов в написании автоматических тестов:
    подготовка тестовых данных,
    вызов продакшн кода,
    проверка результата.
