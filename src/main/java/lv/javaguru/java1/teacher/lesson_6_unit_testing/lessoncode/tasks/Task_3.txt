Написать автоматические тесты на класс MaxNumber метод:

    int findMax(int number1, int number2) {...}

Для этого создайте класс MaxNumberTest и добавьте в него
необходимые тестовые методы.


Цели:
- закрепить навык написания автоматических тестов;
- тренировка в генерации тестовых вариантов;
- повторение трёх шагов в написании автоматических тестов:
    подготовка тестовых данных,
    вызов продакшн кода,
    проверка результата.
