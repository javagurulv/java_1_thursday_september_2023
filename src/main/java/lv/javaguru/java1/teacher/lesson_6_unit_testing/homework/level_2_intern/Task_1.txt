Написать автоматические тесты на класс PalindromeChecker
в котором реализован метод определения является
ли строка палиндромом или нет.

Класс PalindromeChecker лежит рядом с этим файлом задания.

Для выполнения задания скопируйте класс PalindromeChecker к себе в пакет.

Создайте класс PalindromeCheckerTest и добавьте в него
необходимые тестовые методы.

Цели:
- изучить сравнение типов данных String (метод equals());
- закрепить навык написания автоматических тестов;
- тренировка в генерации тестовых вариантов;
- повторение трёх шагов в написании автоматических тестов:
    подготовка тестовых данных,
    вызов продакшн кода,
    проверка результата.
