Спроектировать решение задачи нахождения корней
квадратного уравнения разложив ее на отдельные методы.

Подсказка: в решении должны присутствовать:
- метод нахождения дискриминанта;
- метод отвечающий на вопрос есть ли корни
  у квадратного уравнения или нет;
- метод нахождения первого корня;
- метод нахождения второго корня.

Реализуйте методы в классе SquareEquation.

Создайте класс SquareEquationApp и в его main() методе
соберите полное решение задачи используя для этого методы
из класса SquareEquation. Параметры квадратного уравнения
можно запрашивать у пользователя через консоль или
использовать заранее заданные их значения в виде переменных
(int a = 10; int b = 2; int c = 5).


Цели:
- продемонстрировать деление задачи на подзадачи;
- продемонстрировать реализацию подзадач в виде методов;
- продемонстрировать использование подзадач при решении целой задачи.
