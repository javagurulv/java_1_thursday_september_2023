Написать алгоритм для расчёта площади треугольника
по формуле Герона. По возможности решение разбить на методы.

Реализуйте решение в виде методов в классе TriangleArea.

Продемонстрируйте использование алгоритма несколько раз.
Для этого создайте класс TriangleAreaApp с main(...) методом,
в main() методе создайте экземпляр класса TriangleArea
и найдите площади нескольких треугольников,
распечатайте на консоль полученные результаты.
