Создать метод для вычисления средней оценки
школьников сдавших экзамен по математике.

Оценки школьников в программе задать массивом целых чисел.

Продемонстрировать работу метода с помощью автоматических тестов.

PS: после написания метода и тестов к нему попробуйте
поискать способ упрощения кода и если найдёте, то выполните
рефакторинг.


Цели:
- познакомиться со структурой данных массив;
- познакомиться со способом создания массива new int[];
- познакомиться с циклом for;
- познакомиться с процедурой рефакторинг.
