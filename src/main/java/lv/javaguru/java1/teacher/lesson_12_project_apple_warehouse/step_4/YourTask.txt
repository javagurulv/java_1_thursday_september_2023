Используя параметры метода, можно управлять его поведением!

Управление поведением метода через его параметры.

Передавая разные параметры мы можем заставить метод
выполнять разные действия.

Например, передавая цвет яблок:
- (green) - метод будет искать зелёные яблоки;
- (red) - метод будет искать красные яблоки.

В AppleWarehouse.findApplesByColor() можно найти реализацию.

Теперь с помощью этого метода вы можете искать яблоки разных цветов
и для этого не надо копировать программный код:
List<Apple> greenApples = findApplesByColor(inventory, "green");
List<Apple> redApples = findApplesByColor(inventory, "red");

С помощью тестов убедитесь, что метод
AppleWarehouse.findApplesByColor() работает правильно.
