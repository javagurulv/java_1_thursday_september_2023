Фермер выдвигает вам очередное требование:
Поиск яблок по цвету и весу вместе или по отдельности.

То есть фермер хочет иметь возможности:
- искать яблоки только по цвету;
- искать яблоки только по весу;
- искать яблоки и по цвету и по весу вместе.

Возможно в будущем у яблок могут появиться новые свойства,
например срок хранения. По этим свойствам фермер тоже захочет
делать поиск сочетая их в разных комбинациях с уже реализованными
свойствами: цвет и вес.


Ваша задача: подумайте/поразмышляйте как можно реализовать
функциональность поиска согласно выдвинутым требованиям.

Начните с того, что проанализируйте из каких двух основных
частей состоит функционал поиска?

Подсказка: обратите внимание на любой из уже реализованных
методов поиска и найдите два оператора for() и if().

За что отвечает оператор for()? (поясните словами)

За что отвечает оператор if()? (поясните словами)

Оказывается, любой поиск состоит из двух частей:
1. перебор всех значений (цикл for());
2. проверка значения при помощи критерия поиска (оператор if()).

Если сравнить код всех уже реализованных нами методов поиска,
мы увидим, что они отличаются только критериями поиска,
условием поиска, кодом внутри оператора if().

Перебор всех значений осуществляется одинаково (цикл for())!

Критерий поиска от метода к методу меняется.

передать любой критерий поиска?
