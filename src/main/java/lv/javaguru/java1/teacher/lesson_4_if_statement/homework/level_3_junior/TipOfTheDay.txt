Операции сравнения в Java используются для сравнения двух значений.
Это основа для создания условий в программировании.

Вот основные операторы сравнения в Java:

1. **Равно (==)**: Этот оператор проверяет, равны ли два значения.
Например, `5 == 5` будет `true`, потому что 5 равно 5.

2. **Не равно (!=)**: Этот оператор проверяет, не равны ли два значения.
Например, `5 != 4` будет `true`, потому что 5 не равно 4.

3. **Больше (>)**: Этот оператор проверяет, больше ли одно значение,
чем другое. Например, `5 > 4` будет `true`, потому что 5 больше 4.

4. **Меньше (<)**: Этот оператор проверяет, меньше ли одно значение,
чем другое. Например, `4 < 5` будет `true`, потому что 4 меньше 5.

5. **Больше или равно (>=)**: Этот оператор проверяет, больше или
равно ли одно значение другому. Например, `5 >= 5` будет `true`,
потому что 5 не меньше 5.

6. **Меньше или равно (<=)**: Этот оператор проверяет, меньше
или равно ли одно значение другому. Например, `4 <= 5` будет `true`,
потому что 4 не больше 5.

Эти операторы можно использовать с любыми числовыми типами данных,
а также с типом `char`, где символы сравниваются по их числовому
представлению в ASCII.

Также, обрати внимание, что для сравнения объектов,
таких как строки (`String`), следует использовать методы,
такие как `equals()`, а не операторы сравнения,
так как операторы сравнивают ссылки на объекты, а не их содержимое.
