Создайте программу, которая принимает целое число
и проверяет, попадает ли оно в диапазон от 1 до 10, включительно.
Если да, программа выводит на консоль сообщение об этом.
Если нет, программа выводит соответствующее сообщение.

Цели:
- знакомство с операторами <= и >=;
- знакомство с оператором AND (&&)
  и объединением простых условий в более сложное.

Набросок решения:
{
    int number = 7;
    if ((number >= 1) && (number <= 10)) {
        System.out.println("Number is inside range (1-10)!");
    } else {
        System.out.println("Number is NOT inside range (1-10)!");
    }
}
